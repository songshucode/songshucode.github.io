hexo Markedown 语法

发表于 2019-05-12 | 更新于 2019-07-10 | 分类于 blog

Markdown 是一种标记语法，使用简单的文本格式来表达复杂的标记语言。
基本的语法介绍。

标题

快捷键： (提高文字等级，降低文字等级)
Ctrl+Shift+[ / ]

常用语法

表达式 描述 快捷键
**txt** 粗体 Ctrl+B
_txt_ 斜体 Ctil+I
_**txt**_ 加粗斜体 Ctrl+B+I
~~txt~~ 删除线 -

引用块

代码块

快捷键：行内 | 多行
Ctrl+M+I | Ctrl+M+C

公式块

快捷键：(Ctrl+M冲突，改为Ctrl+Alt+M)
Ctrl+Alt+M
更多公式语法

分割线

列表

快捷键：有序列表 | 无序列表
Ctrl+M+1 | Ctrl+M+B

表格

快捷键：
Ctrl+Alt+T
在表头下添加：可以进行左对齐，右对齐，中间对齐

超链接

快捷键：
Ctrl+L + url

图片

使用 hexo-asset-image 插件可以在创建文章时生成一个同名的文件夹用来存储图片
快捷键：
Ctrl+Shift+L + url

模板设置

在/scaffolds/post.md中可以修改新建文章的模板
