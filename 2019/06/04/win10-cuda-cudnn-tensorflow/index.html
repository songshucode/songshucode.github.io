win10-cuda-cudnn-tensorflow

发表于 2019-06-04 | 更新于 2019-06-10

install cuda10.1,cudnn7.6,tensorflow1.13.1
note: It’s working.

install CUDA

  1. download the cuda filecuda_10.1.168_425.25_win10.exe from the CUDA official website
  2. select the default installing option
  3. set the environment variables

install cudnn

  1. downloading the cudnn filecudnn-10.1-windows10-x64-v7.6.0.64.zipfrom the CUDA official website
  2. copy the cudnn files to the corresponding CUDA folder.

install Anaconda

  1. downloading the Anaconda installation file for the Anacoada official website

  2. set the environment variables

  3. create python3.7 working environment

  4. download the tensorflow-gpu install file tensorflow_gpu-1.13.1-cp37-cp37m-win_amd64.whl

  5. install this file under the working environment

  6. ok, enjoy it.
